跨设备布局

不要认为你的用户更偏好某种设备，确保任何他们选择的设备都能获取出色的体验。响应式WEB设计的主要目标：创建能适配所有设备的站点、应用和体验。

3

参考指南

  1. 响应式WEB设计基础

    大多数WEB没有针对那些跨屏幕体验来做优化。学习基础知识来让你的站点能支持移动电话、桌面或任何单屏设备。

  2. 响应式WEB设计模式

    响应式WEB设计模式在迅速地演变中，但这里有一小部分已确立的模式能顺利运行在桌面和移动设备上。

  3. 导航和动作模式

    学习如何整合导航模式到你的站点上，并让用户们完成你希望他们去做的动作。
